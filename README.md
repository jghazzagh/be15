# POFT 1301 – Business English Interactive Syllabus

McLennan Community College · Spring 2026 · Online

## Overview

An interactive HTML/CSS/JavaScript syllabus for POFT 1301 (Business English), built as a static site with no dependencies beyond a Google Fonts import.

## Files

- `index.html` — Main page structure and content
- `style.css` — All styles (IBM Plex Mono + Sans typefaces, navy/amber palette)
- `app.js` — Interactivity: scroll spy, accordion, grade bar animation, schedule filtering

## Features

- Sticky sidebar navigation with active section highlighting
- Live schedule table with automatic week status detection (past/current/upcoming/break)
- Filterable schedule (All / Upcoming / Breaks)
- Accordion for policies section
- Animated grade breakdown bars
- Responsive layout with mobile hamburger nav
- No frameworks, no build tools — just open `index.html`

## GitHub Pages Deployment

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, root (`/`) folder
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`

## Local Preview

Just open `index.html` in any browser. No server required.
